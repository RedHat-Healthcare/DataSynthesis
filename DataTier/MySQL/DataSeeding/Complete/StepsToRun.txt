
The scripts for simplicity are hard coded. You will need to change the core directory to run the script

General
-------
1. Start MySQL Command Line Client
2. run: mysql -u root -p 
3. Enter password to get to mysql> prompt
4. source Base-DataLoader-DataSynthesis-<Latest>.sql

Mac
---

1. Command Line
2. sudo /Applications/MySQLWorkbench.app/Contents/MacOS/mysql -u root -p
3. cd /Users/mac-beast/Development/DataLoads/DataSeeding
4. source Base-DataLoader-DataSynthesis-<Latest>-Linux.sql
